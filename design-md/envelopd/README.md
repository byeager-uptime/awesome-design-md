# Envelopd (Pocket Stack)

Personal finance envelope budgeting app. Warm, tactile, card-first mobile UI.

- **Style**: Warm earth tones, generous rounding, stacked card metaphor
- **Accent**: Vibrant orange (`#F0920D`)
- **Fonts**: Instrument Serif (display), DM Sans (body)
- **Shadows**: Warm brown-tinted, multi-layer
